Diversos proyectos donde se trabaja con análisis de datos y elementos estadísticos tanto descriptivos como inferenciales.

Cada carpeta cuenta con un PDF de reporte técnico y código usado para la limpieza y depuración de las bases de datos, 
calculos estadísticos y visualización de datos.

Analisis E-commerce: se visualiza comportamiento de regiones de ventas de productos electrónicos y con una prueba de homogeneidad se concluye
que las regiones son negocios solidos y consolidados en el comportamiento de las 4 regiones, con áreas de oportunidad en una más que otra.

Inferencia Vehículos: Se realiza inferencia en las características de dos tipos de vehículos: automóviles y camionetas. 

Sucidios Mérida, Yucatán: Se realiza limpieza y depuración de datos de suicidios de México para enfatizar en Yucatán, se obtienen
resultados e insights interesantes en cuánto el comportamiento de estos datos.

Regresión Lineal Simple: Se analizan 3 variables explicativas en relación a una variable de respuesta. 
Se obtiene un modelo óptimo. Se justifica la elección y decisión.
